# Лабораторная работа 2

## Вариант на тройку

В данной работе используется локальный сервер Ollama и модель Qwen2.5:0.5b. Цель работы — отправить запросы к LLM по HTTP, получить ответы модели и сохранить результаты инференса в таблицу.

## Стек

- Ollama
- Qwen2.5:0.5b
- Python
- requests
- pandas
- Jupyter Notebook

## Установка

Сначала необходимо склонировать репозиторий и перейти в папку с лабораторной работой:

```bash
git clone https://github.com/Goida88/CS.git
cd "CS/lab2 NLP"
```
## Обновить pip и установить необходимые библиотеки
```
python -m pip install --upgrade pip
python -m pip install requests pandas notebook jupyter ipykernel
```
## Усли Ollama еще не установлен нужно скачать модель
```
ollama pull qwen2.5:0.5b
```
Проверить что модель загружена
```
ollama list
```
## Если локальный сервер Ollama не запущен автоматически, его можно запустить командой:
```
ollama serve
```
## Что содержится в ноутбуке
Ноутбук включает:

1. подключение библиотек и настройку параметров;
2. проверку доступности локального сервера Ollama;
3. проверку списка загруженных моделей;
4. выполнение одного тестового запроса;
5. массовый прогон 10 запросов через модель;
6. сохранение результатов в CSV.

# Описание работы
В лабораторной работе Python-скрипт отправляет HTTP-запросы на локальный сервер Ollama по адресу:
```
http://localhost:11434/api/generate
```
В запросе указывается название модели:
```
qwen2.5:0.5b
```
Также передаётся текстовый запрос к модели. После обработки запроса модель возвращает ответ, который сохраняется в таблицу.

Для выполнения запросов используется библиотека requests, а для формирования таблицы используется библиотека pandas.

## Результат
В результате работы формируется отчёт инференса со следующими полями:
1. номер запроса;
2. запрос к LLM;
3. соответствующий ответ модели.
Итоговый отчёт сохраняется в файл:
```
inference_report.csv
```
